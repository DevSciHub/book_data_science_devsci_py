3.1 Tuplas em Python

Em Python, uma tupla é uma estrutura de dados semelhante a uma lista, mas com uma diferença fundamental: as tuplas são imutáveis, uma vez que você criar uma tupla não poderá alterar seus elementos ou adicionar/remover elementos. As tuplas são definidas por uma sequência ordenada de elementos entre parênteses e separados por vírgulas.

tickers_tupla = ('TOTS3.SA', '^BVSP', 'PETR3.SA', 'MGLU3.SA', 'HAPV3.SA')
comprimento_tupla = len(tickers_tupla)
print("Tupla:", tickers_tupla, "\n",
      "Classe:", type(tickers_tupla), "\n",
      "Comprimento:", comprimento_tupla, "elementos")
Tupla: ('TOTS3.SA', '^BVSP', 'PETR3.SA', 'MGLU3.SA', 'HAPV3.SA') 
 Classe: <class 'tuple'> 
 Comprimento: 5 elementos

3.1.1 Características

1 - Imutabilidade: Você não pode modificar os elementos de uma tupla após criá-la. Isso as torna úteis para representar dados que não devem ser alterados acidentalmente;

try: 
  tickers_tupla.append('BBDC4.SA')
except:
  traceback_str = traceback.format_exc()
  print(traceback_str)

    tickers_tupla.append('BBDC4.SA')
    ^^^^^^^^^^^^^^^^^^^^
AttributeError: 'tuple' object has no attribute 'append'

2 - Indexação: Assim como as listas, os elementos de uma tupla podem ser acessados por meio de índices. O índice começa em 0 para o primeiro elemento;

print("1º Elemento:", tickers_tupla[0], "\n",
      "2º Elemento:", tickers_tupla[1], "\n",
      "3º Elemento:", tickers_tupla[-3], "\n")
1º Elemento: TOTS3.SA 
 2º Elemento: ^BVSP 
 3º Elemento: PETR3.SA

3 - Comprimento Fixo: Uma vez que você cria uma tupla com um determinado número de elementos, esse número não pode ser alterado. Você não pode adicionar ou remover elementos;

4 - Uso de Parênteses Opcionais: Embora seja comum usar parênteses para criar uma tupla, eles não são estritamente necessários. Os elementos podem ser separados apenas por vírgulas, mas os parênteses podem ajudar a melhorar a clareza do código.

tickers_tupla = 'TOTS3.SA', '^BVSP', 'PETR3.SA', 'MGLU3.SA', 'HAPV3.SA'
comprimento_tupla = len(tickers_tupla)
print("Tupla:", tickers_tupla, "\n",
      "Classe:", type(tickers_tupla), "\n",
      "Comprimento:", comprimento_tupla, "elementos")

 Classe: <class 'tuple'> 
 Comprimento: 5 elementos

3.2 Listas

Uma lista é uma estrutura de dados que permite armazenar uma coleção ordenada de elementos. Esses elementos podem ser de qualquer tipo de dado, como números, strings, booleanos e até outras listas. As listas são mutáveis, o que significa que você pode adicionar, remover ou modificar elementos após a criação da lista.

3.2.1 Caraterísticas

1 - Mutabilidade: Ao contrário das tuplas, você pode modificar, adicionar ou remover elementos de uma lista após criá-la.

tickers_lista = ['TOTS3.SA', '^BVSP', 'PETR3.SA', 'MGLU3.SA', 'HAPV3.SA']
print("Lista:", tickers_lista, "\n",
      "Classe:", type(tickers_lista), "\n",
      "Comprimento:", len(tickers_lista), "elementos")
Lista: ['TOTS3.SA', '^BVSP', 'PETR3.SA', 'MGLU3.SA', 'HAPV3.SA'] 
 Classe: <class 'list'> 
 Comprimento: 5 elementos

2 - Indexação: Os elementos de uma lista podem ser acessados usando índices. O índice começa em 0 para o primeiro elemento.

print("1º Elemento:", tickers_lista[0], "\n",
      "2º Elemento:", tickers_lista[1], "\n",
      "3º Elemento:", tickers_lista[-3], "\n")
1º Elemento: TOTS3.SA 
 2º Elemento: ^BVSP 
 3º Elemento: PETR3.SA

3 - Comprimento Variável: Você pode adicionar ou remover elementos de uma lista a qualquer momento, alterando o seu comprimento.

try: 
  tickers_lista.append('BRSR6.SA')
except:
  traceback_str = traceback.format_exc()
  print(traceback_str)

print("Lista:", tickers_lista, "\n",
      "Classe:", type(tickers_lista), "\n",
      "Comprimento:", len(tickers_lista), "elementos")
Lista: ['TOTS3.SA', '^BVSP', 'PETR3.SA', 'MGLU3.SA', 'HAPV3.SA', 'BRSR6.SA'] 
 Classe: <class 'list'> 
 Comprimento: 6 elementos

3.2.2 Tuplas e listas

Utilizar uma tupla depois transformar em lista.

tickers_tupla = ('TOTS3.SA', '^BVSP', 'PETR3.SA', 'MGLU3.SA', 'HAPV3.SA')
tickers = list(tickers_tupla)
print(tickers, "\n", "Classe:", type(tickers))
['TOTS3.SA', '^BVSP', 'PETR3.SA', 'MGLU3.SA', 'HAPV3.SA'] 
 Classe: <class 'list'>

3.3 Conjuntos

Em Python, um conjunto (set) é uma coleção não ordenada e mutável de elementos únicos. Os conjuntos são definidos usando chaves {} e podem conter elementos de diferentes tipos de dados. A principal característica dos conjuntos é que eles não aceitam elementos duplicados.

3.3.1 Caraterísticas

1 - Coleção Não Ordenada: Os elementos em um conjunto não têm uma ordem específica. Quando você itera sobre um conjunto, a ordem em que os elementos são retornados pode variar.

elementos_conjunto = {1, 3, 5, 2}
print("Conjunto:", elementos_conjunto, "\n",
      "Classe:", type(elementos_conjunto), "\n",
      "Comprimento:", len(elementos_conjunto), "elementos")
Conjunto: {1, 2, 3, 5} 
 Classe: <class 'set'> 
 Comprimento: 4 elementos

2 - Elementos Únicos: Um conjunto não pode conter elementos duplicados. Se você tentar adicionar um elemento que já está presente no conjunto, a segunda tentativa será ignorada.

tickers_conjunto = {'TOTS3.SA', '^BVSP', 'PETR3.SA', 'MGLU3.SA', 'HAPV3.SA', 'HAPV3.SA'}
print("Conjunto:", tickers_conjunto, "\n",
      "Classe:", type(tickers_conjunto), "\n",
      "Comprimento:", len(tickers_conjunto), "elementos")
Conjunto: {'^BVSP', 'HAPV3.SA', 'PETR3.SA', 'TOTS3.SA', 'MGLU3.SA'} 
 Classe: <class 'set'> 
 Comprimento: 5 elementos

3 - Mutabilidade: Os conjuntos são mutáveis, o que significa que você pode adicionar e remover elementos após a criação do conjunto. No entanto, os próprios conjuntos são objetos mutáveis.

try: 
  tickers_conjunto.add('BBDC4.SA')
except:
  traceback_str = traceback.format_exc()
  print(traceback_str)
  
print("Lista:", tickers_conjunto, "\n",
      "Classe:", type(tickers_conjunto), "\n",
      "Comprimento:", len(tickers_conjunto), "elementos")
Lista: {'^BVSP', 'HAPV3.SA', 'PETR3.SA', 'BBDC4.SA', 'TOTS3.SA', 'MGLU3.SA'} 
 Classe: <class 'set'> 
 Comprimento: 6 elementos

Os conjuntos fornecem métodos úteis para adicionar elementos (add()), remover elementos (remove(), discard()), verificar a existência de um elemento (in), limpar todos os elementos (clear()), entre outros.

4 - Imutabilidade de Elementos: Embora os conjuntos em si sejam mutáveis, os elementos contidos no conjunto devem ser imutáveis. Por exemplo, você pode ter strings, números ou tuplas como elementos de um conjunto, mas não pode ter listas, que são mutáveis.

try: 
  tickers_conjunto = {tickers_lista}
except:
  traceback_str = traceback.format_exc()
  print(traceback_str)

5 - Iterável: Você pode iterar facilmente sobre os elementos de um conjunto usando um loop for (cenas para próxima aula).

3.4 Dicionários

Em Python, um dicionário é uma estrutura de dados que permite armazenar pares chave-valor. Ele é semelhante a uma lista, mas em vez de acessar elementos por índices numéricos, você os acessa por chaves. Dicionários são implementados como tabelas de dispersão, proporcionando acesso rápido e eficiente aos valores associados a cada chave.

3.4.1 Caraterísticas

1 - Chaves Únicas: Cada chave em um dicionário é única, o que significa que não pode haver chaves duplicadas. Se você tentar adicionar uma chave que já existe, o valor associado à chave será atualizado.

2 - Sintaxe de Chaves e Dois-Pontos: A sintaxe para definir um dicionário usa chaves ({}) e dois-pontos (:) para indicar os pares chave-valor. Por exemplo: meu_dicionario = {‘chave1’: ‘valor1’, ‘chave2’: Valor2}.

tickers_dict = {"Ticker": "BBDC4.SA", "Nome da Empresa": "Banrisul"}
print("Dicionário:", tickers_dict, "\n",
      "Classe:", type(tickers_dict), "\n",
      "Comprimento:", len(tickers_dict), "elementos")
Dicionário: {'Ticker': 'BBDC4.SA', 'Nome da Empresa': 'Banrisul'} 
 Classe: <class 'dict'> 
 Comprimento: 2 elementos

3 - Acesso por Chave: Você acessa os valores em um dicionário usando suas chaves.

print("1º Elemento:", tickers_dict["Ticker"], "\n",
      "2º Elemento:", tickers_dict["Nome da Empresa"])
1º Elemento: BBDC4.SA 
 2º Elemento: Banrisul
